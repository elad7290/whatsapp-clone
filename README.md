# Whatsapp Project By Elad & Chen

## Installations required :

* node.js
* npm
* react-router-dom

## How to run our project ?

In the project directory, you need to run:

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.
